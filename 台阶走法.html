一个长阶梯有n级，可以一次走1级，一次走2级，一共有多少种走法？
            递归实现. 
 
            重要的是理解这个逻辑 
            
            假设有f(n)种走法， 
            
            当走到N-1阶台阶时，有f(n-1)种走法，再走一步走完。 
           
             当走到n-2阶台阶时，有f(n-2)种走法， 再走1+1或2，走完。其中走1+1和走到n-1阶时的f(n-1)重复。 
           
             所以f(n)=f(n-1))+f(n-2) 
            
            
           public int findStep(int n) {
                   if(n==0 || n==1 || n==2) {
                       return n;
                   }
                   
                   return findStep(n-1)+findStep(n-2);
               }
